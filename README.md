# 📊 نظام إدارة عملاء الشركة

نظام متقدم وآمن لإدارة بيانات عملاء شركتك مع قاعدة بيانات PostgreSQL قوية.

## ✨ المميزات

✅ **واجهة جميلة وسهلة الاستخدام** - ديزاين احترافي باللغة العربية
✅ **قاعدة بيانات آمنة** - جميع البيانات عندك أنتِ فقط
✅ **إدارة كاملة** - إضافة، تعديل، حذف، بحث عن العملاء
✅ **إحصائيات فورية** - إجمالي العملاء والإحصائيات الديموغرافية
✅ **API قوي** - يمكن دمجه مع أي تطبيق آخر
✅ **البيانات المطلوبة:**
  - الاسم الأول والثاني
  - المكان / العنوان
  - رقم الهاتف
  - النوع (ذكر/أنثى - اختياري)

---

## 📋 متطلبات النظام

قبل البدء، تأكدي من تثبيت:

1. **Node.js** (v14 أو أحدث)
   - تحميل من: https://nodejs.org/
   
2. **PostgreSQL** (قاعدة البيانات)
   - تحميل من: https://www.postgresql.org/download/

3. **Git** (اختياري)

## 🚀 خطوات التركيب والتشغيل

### الخطوة 1️⃣: إنشاء قاعدة البيانات

افتحي **PostgreSQL** وشغلي الأوامر الموجودة في ملف `database.sql`:

```sql
-- انسخي كل الأوامر من ملف database.sql
-- والصقيها في PostgreSQL
```

أو عن طريق سطر الأوامر:

```bash
psql -U postgres -f database.sql
```

---

### الخطوة 2️⃣: إعداد المشروع

1. **انسخي جميع الملفات** إلى مجلد على جهازك:
```
company-clients-system/
├── server.js
├── index.html
├── package.json
├── .env
└── database.sql
```

2. **افتحي terminal/command prompt** في المجلد

3. **ثبتي المكتبات المطلوبة:**
```bash
npm install
```

---

### الخطوة 3️⃣: تعديل إعدادات قاعدة البيانات

افتحي ملف `.env` وعدلي البيانات حسب إعدادات PostgreSQL لديك:

```
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres              # ⚠️ غيري كلمة المرور هنا
DB_PASSWORD=password          # ⚠️ ضعي كلمة مرور PostgreSQL الخاصة بك
DB_NAME=company_clients
PORT=3000
```

---

### الخطوة 4️⃣: تشغيل الخادم

في terminal اكتبي:

```bash
npm start
```

يجب تشوفي الرسالة:
```
✅ الخادم يعمل على: http://localhost:3000
```

---

### الخطوة 5️⃣: فتح الصفحة

افتحي المتصفح واكتبي:

```
http://localhost:3000
```

**هسع أنتِ جاهزة تبدأي تضيفي العملاء!** 🎉

## 📱 كيفية الاستخدام

### إضافة عميل جديد

1. اكتبي **الاسم الأول والثاني**
2. ادخلي **المكان** (المدينة أو العنوان الكامل)
3. ادخلي **رقم الهاتف** (مثال: 01001234567)
4. اختاري **النوع** (اختياري)
5. اضغطي **"حفظ البيانات"**

✅ البيانات محفوظة تلقائياً في قاعدة البيانات!

### البحث عن عميل

ادخلي الاسم أو رقم الهاتف في خانة البحث - الصفحة تصفي النتائج تلقائياً ✨

### حذف عميل

اضغطي على زر "حذف 🗑️" بجانب اسم العميل

### عرض الإحصائيات

الإحصائيات تُحدّث تلقائياً كل 10 ثواني:
- إجمالي عدد العملاء
- عدد الذكور والإناث

## 🔧 API Endpoints (للمطورين)

إذا كنتِ تريدين دمج النظام مع تطبيق آخر:

### ✅ إضافة عميل جديد
```bash
POST /api/clients
Content-Type: application/json

{
  "first_name": "أحمد",
  "last_name": "محمد",
  "location": "القاهرة",
  "phone": "01001234567",
  "gender": "ذكر"
}
```

### ✅ الحصول على جميع العملاء
```bash
GET /api/clients
```

### ✅ الحصول على عميل واحد
```bash
GET /api/clients/1
```

### ✅ تعديل بيانات عميل
```bash
PUT /api/clients/1
Content-Type: application/json

{
  "first_name": "محمد",
  "last_name": "علي",
  "location": "الجيزة",
  "phone": "01101234567",
  "gender": "ذكر"
}
```

### ✅ حذف عميل
```bash
DELETE /api/clients/1
```

### ✅ الحصول على الإحصائيات
```bash
GET /api/stats
```

## ⚠️ حل المشاكل الشائعة

### المشكلة: "خطأ في الاتصال بالخادم"

**الحل:**
1. تأكدي من تشغيل الخادم: `npm start`
2. تأكدي من فتح المتصفح على: `http://localhost:3000`
3. افتحي console المتصفح (F12) وشوفي الأخطاء

### المشكلة: "خطأ في قاعدة البيانات"

**الحل:**
1. تأكدي من تشغيل PostgreSQL
2. تأكدي من صحة البيانات في ملف `.env`
3. تأكدي من إنشاء قاعدة البيانات بالأوامر في `database.sql`

### المشكلة: "رقم الهاتف يقول مكرر"

**الحل:**
هذا حماية من قاعدة البيانات - لا يمكن إضافة رقم هاتف مرتين. شيكي إذا كان العميل موجود بالفعل.
